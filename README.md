## ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝรถ๏ฟฝ๏ฟฝัก

- ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสผ๏ฟฝาน๏ฟฝ๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ (PHPMailer + reset token)
- แดช๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝยก๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาท: Student / Teacher / Admin
- ๏ฟฝัด๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝรง๏ฟฝาน, ๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ (tasks), ๏ฟฝ๏ฟฝะติด๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝืบหน๏ฟฝ๏ฟฝ
- ๏ฟฝ่งงานแนบ๏ฟฝ๏ฟฝ๏ฟฝ, ๏ฟฝ๏ฟฝวจ๏ฟฝาน (approve/reject), ๏ฟฝ๏ฟฝะบัน๏ฟฝึก๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัติก๏ฟฝ๏ฟฝ๏ฟฝ่งก๏ฟฝับ๏ฟฝาน
- ๏ฟฝัด๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาชิก๏ฟฝ๏ฟฝรง๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝิญ๏ฟฝาจ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝึก๏ฟฝ๏ฟฝ
- ๏ฟฝะบ๏ฟฝ๏ฟฝ๏ฟฝะก๏ฟฝ๏ฟฝ, ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอน (notifications), ๏ฟฝ๏ฟฝ๏ฟฝ deadline reminder
- หน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝยงาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอก CSV ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝับ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝะบ๏ฟฝ
- ๏ฟฝะบ๏ฟฝ Audit Logs ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝิท๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝยของ Admin
- ๏ฟฝ๏ฟฝ CSRF protection ในฟ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝำคัญ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัป๏ฟฝ๏ฟฝลด๏ฟฝ๏ฟฝ๏ฟฝแบบ๏ฟฝ๏ฟฝอด๏ฟฝ๏ฟฝ๏ฟฝ

## สถาน๏ฟฝสถาปัตยก๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัจ๏ฟฝุบัน

๏ฟฝ๏ฟฝเจก๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาน๏ฟฝ๏ฟฝิงแบบ Hybrid:

- `frontend/public/*.php` ๏ฟฝ๏ฟฝ public entry points
- ๏ฟฝ๏ฟฝ runtime ๏ฟฝัจ๏ฟฝุบัน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `backend/src/Legacy/**`
- ๏ฟฝ๏ฟฝ scaffold ๏ฟฝรง๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `backend/src/{Controllers,Services,Repositories,Views,...}` ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอง๏ฟฝับ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝแฟก๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

๏ฟฝ๏ฟฝุป: ๏ฟฝะบ๏ฟฝ๏ฟฝัจ๏ฟฝุบัน๏ฟฝัง๏ฟฝัน๏ฟฝ๏ฟฝ Legacy ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัก ๏ฟฝ๏ฟฝะก๏ฟฝ๏ฟฝัง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝรง๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

## ๏ฟฝรง๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝเจก๏ฟฝ๏ฟฝ

```txt
rmutp_project/
|- frontend/
|  |- public/                    # public document root
|     |- *.php                   # wrappers -> backend/src/Legacy/*
|     |- Image/
|     |- assets/
|     |- uploads/
|
|- backend/
|  |- libs/PHPMailer/            # mail library
|  |- src/
|  |  |- Legacy/                 # runtime logic ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาน๏ฟฝ๏ฟฝิง
|  |  |- Config/Core/...         # scaffold ๏ฟฝรง๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
|  |- storage/
|
|- docs/
|  |- sql/rmutp_database.sql     # schema + incremental upgrade (๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ)
|
|- buildDatabase.bat             # setup/upgrade ๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝลผ๏ฟฝาน CLI
|- buildAdmin.bat                # ๏ฟฝ๏ฟฝ๏ฟฝาง/๏ฟฝัปเดต admin ๏ฟฝ๏ฟฝาน CLI
|- .htaccess                     # route root -> frontend/public
```

## ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝอง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝะบ๏ฟฝ

- Windows + XAMPP (Apache + MySQL/MariaDB)
- PHP CLI (๏ฟฝะน๏ฟฝ PHP 8+ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `pdo_mysql`)
- MySQL 8+ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ MariaDB 10.4+

## ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาน (Windows + XAMPP)

1. ๏ฟฝิด Apache ๏ฟฝ๏ฟฝ๏ฟฝ MySQL ๏ฟฝ XAMPP
2. ๏ฟฝาง๏ฟฝ๏ฟฝเจก๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `htdocs`
3. ๏ฟฝ๏ฟฝ้งค๏ฟฝ๏ฟฝ DB (๏ฟฝ๏ฟฝาต๏ฟฝอง๏ฟฝ๏ฟฝ๏ฟฝ) ๏ฟฝ๏ฟฝาน environment variables ๏ฟฝ๏ฟฝ `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASS`
4. ๏ฟฝ๏ฟฝ๏ฟฝาง/๏ฟฝัป๏ฟฝรด๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

```bat
buildDatabase.bat
```

5. ๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสผ๏ฟฝาน admin

```bat
buildAdmin.bat --email=admin@rmutp.ac.th --name="Super Admin" --password="StrongPassword123!"
```

6. ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ็บผ๏ฟฝาน URL ๏ฟฝอง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝเจก๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ

```txt
http://localhost/<your-project-folder>/
```

๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาตรง:

```txt
http://localhost/<your-project-folder>/frontend/public/
```

## สค๏ฟฝิป๏ฟฝ๏ฟฝ CLI

### `buildDatabase.bat`

๏ฟฝ๏ฟฝ๏ฟฝยก `backend/src/Legacy/Admin/buildDatabase.php` ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ apply SQL

- SQL ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ: `docs/sql/rmutp_database.sql`
- ๏ฟฝอง๏ฟฝับ options:
  - `--host`
  - `--port`
  - `--database`
  - `--user`
  - `--password`
  - `--sql`

๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาง:

```bat
buildDatabase.bat --database=rmutp --user=root --password=your_password
buildDatabase.bat --sql=docs\sql\rmutp_database.sql
```

### `buildAdmin.bat`

๏ฟฝ๏ฟฝ๏ฟฝยก `backend/src/Legacy/Admin/buildAdmin.php` ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัปเดต๏ฟฝัญ๏ฟฝ๏ฟฝ๏ฟฝอด๏ฟฝิน

- ๏ฟฝอง๏ฟฝับ options:
  - `--email`
  - `--name`
  - `--password`

๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝหต๏ฟฝ:
- ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ `--password` ๏ฟฝะบ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝสผ๏ฟฝาน๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัต๏ฟฝ๏ฟฝัต๏ฟฝ
- ๏ฟฝ๏ฟฝ๏ฟฝสผ๏ฟฝาน๏ฟฝ๏ฟฝอง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝาง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ 10 ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝัก๏ฟฝ๏ฟฝ

| Variable | Default |
|---|---|

| `DB_NAME` | `rmutp` |
| `DB_USER` | `root` |

| Variable | Default |
|---|---|
| `SMTP_HOST` | `smtp.gmail.com` |
| `SMTP_PORT` | `587` |
| `SMTP_ENCRYPTION` | `tls` |
| `SMTP_USER` | `` |
| `SMTP_PASS` | `` |
| `SMTP_FROM_NAME` | `RMUTP Support Team` |
| `APP_BASE_URL` | `` (auto detect) |

### Admin Bootstrap (optional)

| Variable | Description |
|---|---|
| `ADMIN_EMAIL` | default email for `buildAdmin` |
| `ADMIN_FULLNAME` | default full name for `buildAdmin` |
| `ADMIN_PASSWORD` | default password for `buildAdmin` |
